# MealsApp 🥘

A modern, responsive recipe explorer for Android that allows users to discover delicious dishes from around the world.

## Features
- 🔍 **Search & Discover:** Easily find recipes by name or browse by category.
- 📋 **Detailed Recipes:** View comprehensive lists of ingredients and step-by-step instructions.
- 🎨 **Material 3 UI:** A clean, modern interface designed with Jetpack Compose.
- ⚡ **Fast & Fluid:** Optimized image loading and smooth navigation.

## Tech Stack
- **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3)
- **Navigation:** Compose Navigation
- **Networking:** [Retrofit](https://square.github.io/retrofit/) & [Moshi](https://github.com/square/moshi)
- **Image Loading:** [Coil](https://coil-kt.github.io/coil/)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Async:** Kotlin Coroutines & Flow

*Built using TheMealDB API.*
